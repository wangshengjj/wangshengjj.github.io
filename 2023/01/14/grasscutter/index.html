某动漫游戏私服Linux端搭建教程

Grasscutter原Github项目地址

一、环境准备

  • Centos7
  • MongoDB
  • JDK17 RPM包
  • Grasscutter服务端[停止维护]
  • 宝塔面板
  • MCSManager面板

二、配置安装环境

1.安装jdk17

下载

wget https://download.oracle.com/java/17/latest/jdk-17_linux-x64_bin.rpm

安装

rpm -ivh jdk-17_linux-x64_bin.rpm

测试

java -version

如果输出以下信息，就证明安装成功

[root@localhost ~]# java -version
java version "17.0.5" 2022-10-18 LTS
Java(TM) SE Runtime Environment (build 17.0.5+9-LTS-191)
Java HotSpot(TM) 64-Bit Server VM (build 17.0.5+9-LTS-191, mixed mode, sharing)

2.防火墙配置(重要)

请前往云服务器提供商管理面板，配置防火墙策略，依次开放宝塔面板端口(请根据宝塔安装完以后提示的端口)、443、22102、27017、23333、24444。

3.安装宝塔面板

yum install -y wget && wget -O install.sh https://download.bt.cn/install/install_6.0.sh && sh install.sh ed8484bec

安装完成后，根据终端提示的url访问即可，终端会给出默认的账号和密码

如果无法访问，请前往防火墙策略开启相应端口

关闭宝塔面板的系统防火墙(重要)

安装完毕后，浏览器访问服务器IP+23333端口

如果无法访问，请检查防火墙策略是否已经开放23333端口和24444端口
